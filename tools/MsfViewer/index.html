MSF / MSS Viewer

Standalone JWS inspector for Metaverse Service Fabric and Metaverse Service Specification files. Verifies the cryptographic signature against the leaf certificate's public key. Chain-to-root validation is not performed (browsers do not expose the OS trust store).

Drop a .msf or .mss file here

Signature verification uses the browser's native Web Crypto API. The public-key fingerprint matches the Sneeze engine's identity derivation (SHA-256 of the leaf certificate's SubjectPublicKeyInfo).
